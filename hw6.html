insurance$smoker <- factor(insurance$smoker)

Research question: Does smoking increase expected medical insurance charges, and by how much?

  1. DONE

  2. Question 1

The linear regression model is

\[ Y = X\beta + \epsilon \]

where:

  • (Y) is the vector of insurance charges.
  • (X) is the matrix of predictors (age, BMI, smoker, and region).
  • () is the vector of regression coefficients.
  • () is the error term.

\[ \text{charges}_i = \beta_0 + \beta_1(\text{age}_i) + \beta_2(\text{bmi}_i) + \beta_3(\text{smoker}_i) + \beta_4(\text{region}_i) + \epsilon_i. \]

The columns of (X) include:

  • Intercept
  • Age
  • BMI
  • Smoker status
  • Region indicator variables

Each coefficient in () represents the expected change in insurance charges associated with its predictor while holding the other variables constant.

  1. The model answers the research question by estimating how each predictor affects insurance charges while holding the other variables constant. The coefficient for smoker directly addresses whether smoking is associated with higher charges. Its value represents the expected difference in insurance charges between smokers and non-smokers, after accounting for age, BMI, and region.

fit <- lm(charges ~ age + bmi + smoker + region,

The coefficient estimates from lm() and ols() are identical because both methods fit the same linear regression model.

The 95% confidence interval for the smoking coefficient from lm() is approximately

\[ (23041.29,\ 24663.68). \]

The ols() function provides additional information such as effect summaries, predictor ranges, and model validation statistics that are not included in the standard lm() output.

Since region has four levels, R creates three indicator variables and uses one level as the reference category. By default, R uses the first level alphabetically as the reference, which is:

levels(insurance$region)
[1] "northeast" "northwest" "southeast" "southwest"

The reference level is northeast.
