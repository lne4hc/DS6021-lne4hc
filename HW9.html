Ten-fold cross-validation

set.seed(42)

K <- 10
fold_id <- sample(rep(1:K, length.out = nrow(nhanes)))

cv_shared <- rep(NA, K)
cv_offset <- rep(NA, K)
cv_interaction <- rep(NA, K)
cv_separate <- rep(NA, K)

for (k in 1:K) {

  train <- nhanes[fold_id != k, ]
  test  <- nhanes[fold_id == k, ]

  m1 <- ols(
    Weight ~ rcs(Height, height_knots),
    data = train
  )

  m2 <- ols(
    Weight ~ rcs(Height, height_knots) + Race1,
    data = train
  )

  m3 <- ols(
    Weight ~ rcs(Height, height_knots) * Race1,
    data = train
  )

  p1 <- predict(m1, test)
  p2 <- predict(m2, test)
  p3 <- predict(m3, test)

  p4 <- rep(NA_real_, nrow(test))

  for (r in race_levels) {

    train_r <- train |> filter(Race1 == r)
    test_rows <- which(test$Race1 == r)

    if (length(test_rows) > 0) {

      m4 <- ols(
        Weight ~ rcs(Height, height_knots),
        data = train_r
      )

      p4[test_rows] <- predict(
        m4,
        test[test_rows, ]
      )
    }
  }

  cv_shared[k] <- mean((p1 - test$Weight)^2)
  cv_offset[k] <- mean((p2 - test$Weight)^2)
  cv_interaction[k] <- mean((p3 - test$Weight)^2)
  cv_separate[k] <- mean((p4 - test$Weight)^2)
}

task1_results <- data.frame(
  Model = c(
    "Fully shared",
    "Shared shape + offset",
    "Full interaction",
    "Fully separate"
  ),
  CV_MSE = c(
    mean(cv_shared),
    mean(cv_offset),
    mean(cv_interaction),
    mean(cv_separate)
  )
)

Task 1 answer

The model with the lowest cross-validated MSE is the model best supported by the data. If two models have very similar MSE values, the simpler model is preferred because the additional complexity provides little practical improvement.

Examine group deviations

There are four deviation coefficients for each race group:

  1. A group intercept deviation
  2. A linear height deviation
  3. The first nonlinear height deviation
  4. The second nonlinear height deviation

The final race group is reconstructed as the negative sum of the other groups because sum-to-zero contrasts require all group deviations to add to zero.

race_main_term <- which(term_labels == "Race1")
race_int_term <- which(
  grepl("rcs\\(Height", term_labels) &
    grepl("Race1", term_labels)
)

race_main_idx <- which(assign_vec[-1] == race_main_term)
race_int_idx <- which(assign_vec[-1] == race_int_term)

group_deviation_matrix <- function(beta_vector) {

  output <- matrix(
    NA_real_,
    nrow = length(race_levels),
    ncol = 4,
    dimnames = list(
      race_levels,
      c("Intercept", "Height", "Height prime", "Height double prime")
    )
  )

  for (j in seq_along(race_main_idx)) {

    interaction_columns <- race_int_idx[
      (3 * (j - 1) + 1):(3 * j)
    ]

    output[j, ] <- beta_vector[
      c(race_main_idx[j], interaction_columns)
    ]
  }

  output[nrow(output), ] <- -colSums(
    output[-nrow(output), , drop = FALSE]
  )

  output
}

Task 3 answer

The ridge model shrinks all race deviations toward zero but usually does not make them exactly zero. The lasso can set individual deviation coefficients to exactly zero. Groups with fewer nonzero coefficients are more strongly pooled toward the average curve. Groups with four nonzero coefficients remain the most distinct.

Task 4: Conclusion

The final model should be selected primarily using cross-validated MSE.

The Task 1 results show whether a fully shared, offset, interacted, or fully separate model predicts best. The ridge and lasso results show whether some group-specific differences can be retained while weaker differences are shrunk toward the average curve.

A regularized model is preferred when it provides predictive performance similar to or better than the full interaction model while avoiding unstable group-specific curves. The amount of shrinkage should not be explained by sample size alone. A smaller group can still remain distinct when its observed relationship differs strongly from the average curve.

Task 5: Transformation matrix

The standard treatment-coded matrix is called (X), and the sum-to-zero-coded matrix is called (Z).

We need the matrix (R) such that

[ Z = XR. ]

nhanes_standard <- nhanes

contrasts(nhanes_standard$Race1) <- contr.treatment(
  nlevels(nhanes_standard$Race1)
)

nhanes_sum <- nhanes

contrasts(nhanes_sum$Race1) <- contr.sum(
  nlevels(nhanes_sum$Race1)
)

X_standard <- model.matrix(
  ~ Race1,
  data = nhanes_standard
)

Z_sum <- model.matrix(
  ~ Race1,
  data = nhanes_sum
)

Because (Z=XR), the transformation matrix is

[ R=(XX){-1}X^Z. ]

R <- solve(
  crossprod(X_standard),
  crossprod(X_standard, Z_sum)
)

rownames(R) <- colnames(X_standard)
colnames(R) <- colnames(Z_sum)
